# 图书馆座位预约管理

## 主要功能
- 扫码打卡：根据微信小程序打开场景+GPS位置信息进行定位打卡 
- 座位预约：根据所选时间进行预约 
- 座位查询：查询座位使用情况
- 管理员端：对违规用户解禁、处罚、设置开放楼层、开放座位等
